# informLocationImage
- タッチ動作によりBeHereNow.pngを自由動かすことができます。
- タッチ完了時に最も近いlocation#へ吸着されます。
- ただしある条件以上の距離の場合には、元いた位置に吸着されます。
- 画面下のLabelに現在BeHereNow.pngが位置している(表示している)location#を表示します。
